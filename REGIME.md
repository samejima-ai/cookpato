### 不確実性スコアの内訳
- イメージの明確さ: SPEC.md が機能×条件粒度で確定、装飾要件まで定量化 → 0
- 既知度: React 18 + Vite + localStorage + PWA、全て確立パターン → 0

### リスクスコアの内訳
- 失敗時の影響: 妻 1 名のみが使用、データ完全ローカル、外部送信なし。端末ローカルなので技術的には他人も使用可能だが、配布は妻 1 名限定。事故時のダメージは個人メモ喪失に限定 → 1

### NFRスコアの内訳（詳細は `nfr-scoring.md`）

- セキュリティ（S）: 個人用途、完全ローカル、認証不要、データ外部送信なし → 0
- コンプライアンス（C）: 個人用途、規制対象外 → 0
- スケーラビリティ（L）: 単一ユーザー、データ増加は月数十エントリ → 0
